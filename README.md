# TEFModLoader 功能

* 在 Android 设备上安装 Terraria 的 MOD 和资源包。
* 支持多种类型的模组和资源包，增强游戏体验。
* 允许玩家轻松管理和切换不同的模组，以适应不同的游戏需求。
* 提供用户界面，简化模组安装过程，无需复杂的手动操作。
* 为玩家提供社区支持和帮助，方便分享和获取模组安装的帮助和建议。

# 原理

* 资源加载使用iohook（LsPatch）
* 模组使用BNM挂钩
